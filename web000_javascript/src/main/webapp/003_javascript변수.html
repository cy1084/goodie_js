변수의 이용

변수의 선언규칙(식별자/identifier)
1. 대소문자를 구분
2. 영문, $, _로 시작
3. 영문, $, _를 포함할 수 있고, 그 외의 특수문자는 허용되지 않음
4. 키워드 및 예약어는 사용이 불가능하다
변수의 범위
1. 전역변수(global): window객체에 포함되는 변수. 다른 함수들에서 공용으로 사용할 수 있다
값이 유지됨
2. 지역변수: 함수나 객체에서 선언되면 실행이 종료되어 사라짐(let, const : ES6문법)
변수의 선언과 저장
1. var/let의 키워드를 사용하여 변수
2. const 키워드를 사용하여 변수
3. javascript에서 변수의 타입은 동적변수로 되어 있다
javascript 타입의 종류
  • 문자(string)
  • 숫자(number)
  • 논리(boolean)
  • 널(object/null)
  • 객체(object/array)
  • 함수(function)
